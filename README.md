

## Boilerplat Nextjs | Tailwind 3 | Storybook


Install the dependencies
```bash
npm install
#or
yarn install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Run the storybook server:

```bash
npm run storybook
# or
yarn storybook
```

Generate build:

```bash
npm run build
# or
yarn build
```

Run application on Production mode:

```bash
npm run start
# or
yarn start
```
